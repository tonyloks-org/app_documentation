# Подключение Avito API

## Что понадобится
- **Client ID** интеграции Avito.
- **Client Secret** интеграции Avito.
- **User ID** — ID основного профиля Avito (главный аккаунт).

## Как получить данные
1. Откройте "Профиль и настройки" -> "Интеграции API" и выпишите **Client ID** и **Client Secret**.  
   ![Где взять Client ID и Client Secret](Данные%20апи.png)
2. Откройте "Профиль и настройки" -> "Управление профилем".
   ![Где посмотреть User ID профиля](Ссылка%20на%20профиль.png)

## Настройка подключения в приложении
1. Задайте название подключения (любое понятное имя, пойдет в поле `AccountName` в таблицах).
2. Введите `Client ID`, `Client Secret` и `User ID`.
3. Сохраните и дождитесь успешной проверки.

## Что дает подключение
- Статистика объявлений (stats/v2) по метрикам Avito.
- Баланс кошелька + CPA-аванс.
- Контент объявлений (список и детали).
